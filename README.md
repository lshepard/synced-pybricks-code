Jahn Version of pybricks-code
=============

This is a forked version of the illustrious [pybricks-code](https://github.com/pybricks/pybricks-code), which is a web app programming for coding LEGO® Powered Up smart hubs using Pybricks MicroPython.

Why Fork
====

We are the Lego Robotics Team at Jahn Elementary School in Chicago (see our [website](https://jahnschool.com/robotics)). We have been using PyBricks since early 2024 with great success. However, the default Pybricks interface stores files locally - which can make it challenging to support multiple students on the same team sharing code.

This version of the app creates a shared code space so that code can be synced to a common server, thus allowing multiple students to collaborate on the same code even if they are using different laptops.
